# Anjima -- UI/UX Designer & Frontend Developer Portfolio

This is a premium, high-fidelity portfolio website built with React, TypeScript, Tailwind CSS, Framer Motion, and Lucide React.

## 🚀 Live Demo
Check out the live website here:
👉 **[https://portfolio-sigma-topaz-isvvx75onu.vercel.app/](https://portfolio-sigma-topaz-isvvx75onu.vercel.app/)**

---

## 🛠️ Tech Stack & Key Dependencies
- **Core**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## ✨ Key Features
- **Hero Avatar**: Custom 3D female creator avatar with mouse-following magnetic interactive drift (utilizes a custom bounding physics component).
- **Dual Scrolling Marquees**: Fast, scroll-synchronized horizontally scrolling preview rows that move in opposite directions.
- **Scroll Reveal Text**: Custom character-by-character text opacity animation revealing the About page copy as you scroll.
- **Staggered Animations**: Staggered entry fades for list items and component structures.
- **Sticky Stacking Cards**: Overlapping client and personal project cards that scale down dynamically as subsequent cards slide over them.
- **Fully Responsive**: Adapts fluidly across mobile, tablet, and desktop viewports using custom tailwind tokens and CSS clamps.
